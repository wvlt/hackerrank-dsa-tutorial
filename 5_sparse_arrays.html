ðŸ“‹ Problem Statement

For each query string, determine how many times it occurs in the list of input strings.

Example:
Input: ["aba", "baba", "aba", "xzxb"]
Queries: ["aba", "xzxb", "ab"]
Results: [2, 1, 0]

ðŸ’¡ Optimized Solution: Hash Map

def matchingStrings(stringList, queries): # Build frequency map: O(n) frequencies = {} for string in stringList: frequencies[string] = frequencies.get(string, 0) + 1 # Lookup each query: O(m) results = [] for query in queries: results.append(frequencies.get(query, 0)) return results

Time: O(n + m) | Space: O(n + m)
